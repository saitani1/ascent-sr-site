## 労務トラブル発生時の緊急労務相談

### 15,000円／初回60分

問題社員、突然の退職、無断欠勤、休職、解雇、ハラスメントなど、急な労務トラブルについて社労士が状況を整理し、会社が取るべき対応をご案内します。

「従業員にどう伝えればよいのか」「この対応に問題はないか」と迷ったときは、会社だけで判断する前にご相談ください。

オンライン、電話またはご来所で対応します。

※即時対応を保証するサービスではありません。対応可能な日時をご案内します。

# 給与計算

## 毎月の給与計算を、正確かつスムーズに

残業代、社会保険料、欠勤控除など、給与計算には専門的な判断が必要です。

社労士が給与計算を担当することで、単なる数字の計算だけでなく、労働時間や賃金制度の問題にも気づきやすくなります。

| 内容    |    料金（税抜） |
| ----- | --------: |
| 基本料金  | 月額12,000円 |
| 給与計算  | 1名につき800円 |
| 勤怠集計  | 1名につき400円 |
| 賞与計算  |  給与計算1か月分 |
| 初期設定料 |  給与計算1か月分 |

**人数の数え方：給与計算に記載している「1名」は、社会保険の加入者数ではなく、給与計算の対象となる実人数を基準とします。**
例：給与計算の対象者が10名の場合
月額12,000円＋8,000円＝月額20,000円

※勤怠データの内容、給与体系、締切日までの日数などにより、料金を調整する場合があります。
※年末調整は含まれません。

# 助成金申請

## 会社の取り組みに合った助成金の活用を支援します

助成金は、申請書を提出するだけで受給できるものではありません。

雇用契約書、就業規則、賃金台帳、出勤簿などの整備状況を確認し、制度の要件に沿って計画から申請まで支援します。

| 契約状況   |   料金（税抜） |
| ------ | -------: |
| 顧問契約あり |  受給額の16％ |
| 顧問契約なし |  受給額の20％ |
| 着手金    | 24,000円～ |
| 不支給の場合 |   成功報酬なし |

含まれるサービス：

* 活用可能性の確認
* 受給要件の確認
* 必要書類のご案内
* 申請スケジュールの管理
* 計画届・支給申請書の作成
* 行政機関からの確認事項への対応

※助成金の受給を保証するものではありません。
※助成金の種類や難易度により、報酬率・着手金が変わる場合があります。
※会社都合による解約や申請中止の場合は、進捗に応じた報酬を申し受ける場合があります。

# 料金についてのご案内

> ※表示料金はすべて税抜です。別途、消費税を申し受けます。
> ※顧問料金の人数区分は、原則として社会保険の加入人数を基準とします。
> ※顧問契約において、社会保険未加入の従業員に関する相談や手続きが多数発生する場合は、実際の従業員数や業務量を踏まえてお見積りします。
> ※事業所数、雇用形態、入退社の頻度、勤怠管理の状況、相談・手続きの量などにより、料金を調整する場合があります。
> ※複雑な労務トラブル、行政対応、年金請求、第三者行為災害などは、顧問契約とは別にお見積りします。
> ※正式な料金は、現在の状況とご希望の支援内容を確認したうえでご案内します。
